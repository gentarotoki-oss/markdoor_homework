# TODO App

シンプルなタスク管理アプリケーションです。

## 技術スタック

- バックエンド: Python (FastAPI)
- フロントエンド: HTML / CSS / JavaScript
- データ保存: JSON ファイル

## セットアップ手順

### 1. 必要なパッケージのインストール
```
pip install fastapi uvicorn
```

### 2. サーバーの起動
```
python -m uvicorn main:app --reload
```

### 3. アクセス

- アプリ画面: http://127.0.0.1:8000/static/index.html
- API ドキュメント: http://127.0.0.1:8000/docs

## 実装済み機能

### バックエンド (API)

- `POST /todos` - タスクの作成（タイトル、詳細、作成日時の自動付与）
- `GET /todos` - 全タスクの取得
- `PUT /todos/{id}` - タスクの更新（タイトル、詳細、完了状態）
- `DELETE /todos/{id}` - タスクの削除

### フロントエンド

- タスクの追加（タイトル＋詳細説明）
- タスク一覧の表示
- フィルタリング（全て / 未完了 / 完了済み）
- 完了・未完了の切り替え（視覚的に取り消し線で区別）
- タスクの削除（確認ダイアログ付き）
- 入力エラー時のアラート表示

## データ保存

タスクデータは `todos.json` にJSON形式で保存されます。
